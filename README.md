# ArtHOI

Project page for **ArtHOI: Articulated Human-Object Interaction Synthesis by 4D Reconstruction from Video Priors**.

## Deploy to arthoi.github.io

1. Create a GitHub repo named `arthoi.github.io`
2. Push this folder to the `main` branch
3. Enable GitHub Pages: Settings → Pages → Source: Deploy from branch `main` / root

## Local Preview

```bash
cd arthoi.github.io
python3 -m http.server 8000
# Open http://localhost:8000
```

## Structure

- `index.html` - Main project page
- `css/` - Bootstrap 4.4.1 (local)
- `js/` - jQuery, Popper, Bootstrap, app.js (local)
- `videos/` - Demo videos (extracted from PPT)
- `assets/` - Teaser, pipeline, comparisons images, bib.txt
- `README.md` - This file
